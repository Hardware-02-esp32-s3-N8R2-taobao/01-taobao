# 3 米 / 10 平方米场景采购建议

## 场景假设

- 人脸距离：约 `3 m`
- 覆盖面积：约 `10 平方米`
- 目标：先验证人脸识别准确性与实时性，再进入追踪

## 推荐方案

### 方案 A：最推荐

- `ESP32-S3` 摄像头开发板，不强制带屏幕
- `OV5640`
- `60° ~ 72°` 视场角镜头

原因：

- 画质和像素余量更好
- 对 `3 m` 识别更友好
- 不额外上屏幕，调试更轻

### 方案 B：低成本验证

- `ESP32-S3`
- `OV2640`
- 普通视角镜头

原因：

- 先验证脸框输出和基本实时性完全够用
- 成本更低

局限：

- 如果后面追求更远距离下的识别准确性，可能很快想升级

## 带屏幕是否值得买

如果你现在最看重：

- 研发效率
- 坐标输出
- 远端网页调试

那就不必为了屏幕多花预算。

如果你现在最看重：

- 脱机展示
- 一体化演示效果

那可以买带屏幕板。

## 不建议

- 超广角镜头
- 鱼眼镜头
- 只因为“看起来高级”就优先选带屏幕方案

## 一句话建议

对你这个场景，最稳的选择是：

`ESP32-S3 + OV5640 + 中等视场角镜头 + 网页预览`
