# 硬件索引

## 这一阶段的最小硬件要求

只为了跑通“人脸坐标输出”，最小只需要：

- `ESP32-S3` 开发板，最好带 `PSRAM`
- 摄像头模组
- USB 数据线

这一阶段先不强制要求：

- 舵机
- 云台支架
- 独立舵机电源

## 推荐硬件路线

### 最稳

- `ESP32-S3-EYE`
- `ESP32-S3-Korvo-2`

原因：

- 官方 `ESP-WHO` 对这两块板的资料最完整。

### 最省钱

- 任意带 `PSRAM` 的 `ESP32-S3 + OV2640/OV5640`

原因：

- 这一步主要验证脸检与坐标输出，不一定非要一步到位买完整云台套件。

## 和 07 的区别

`08` 阶段主要看：

- 摄像头是否稳定
- `PSRAM` 是否够用
- 人脸框坐标能不能稳定输出

`07` 阶段才要重点看：

- GPIO 是否方便接舵机
- 电源是否足够
- 结构是否适合做云台
