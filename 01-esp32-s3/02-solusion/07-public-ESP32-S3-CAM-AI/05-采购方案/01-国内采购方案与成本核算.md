# 国内采购方案与成本核算

更新时间：`2026-03-16`

这份清单按“尽快落地”来排，不按“参数最好看”来排。

## 先给结论

如果你想尽快开始并且少踩坑，我建议优先级如下：

1. `方案 B`：微雪 `ESP32-S3 AI 摄像头开发板` + 双舵机云台
2. `方案 A`：`ESP32-S3-EYE` + 双舵机云台
3. `方案 C`：`XIAO ESP32-S3 Sense` + `OV5640` + 双舵机云台
4. `方案 D`：`ESP32-S3-Korvo-2` + 双舵机云台

原因很直接：

- `方案 B` 国内买最方便，成本最稳，资料也比较全。
- `方案 A` 官方路线最纯，和 `ESP-WHO` 对齐最好。
- `方案 C` 体积最小，适合以后塞进成品结构。
- `方案 D` 硬件最豪华，但成本和采购不确定性都最高。

## 统一默认配件

为了方便你直接采购，下面所有方案都默认加上这些：

- 二自由度舵机云台支架 × 1
- `MG90S` 金属齿轮舵机 × 2
- 稳定 `5V 3A` 电源 × 1
- 杜邦线 / 端子线若干

说明：

- 双舵机做 `pan + tilt` 足够完成“人脸跟随”演示。
- 舵机建议单独稳压供电，并和开发板共地。

## 风险提醒

- 淘宝同名板卡存在很多兼容版和改版，购买前要确认：
  - 是否带 `PSRAM`
  - 摄像头型号
  - 是否带屏幕 / 麦克风
  - 是否提供 `ESP-IDF` 或 `Arduino` 示例
- 舵机不要直接赌板载 `5V` 口长期稳定，演示能亮不代表长期可靠。
- `ESP32-S3-Korvo-2` 和 `ESP32-S3-EYE` 的购买渠道价格波动较大，下单前建议再看一次库存。
