# ESP32 太阳能低功耗管理参考

这个目录专门整理 `ESP32 + 太阳能板 + 锂电池 + 低功耗采集上传` 的项目方案。

当前目标场景：

- 每 `10` 分钟采集一次数据
- 通过网络上传一次
- 监测电池电压、电量、电流
- 由太阳能板给电池补能

目录说明：

- `01-需求与结论`：先看这里，快速判断当前板卡是否合适、推荐怎么做。
- `02-系统架构`：电源、采集、上传、休眠的系统架构。
- `03-功耗预算`：10 分钟周期下的平均电流、太阳能板和电池容量估算。
- `04-采购方案`：模块购买链接、价格、总价和推荐组合。
- `05-成熟方案`：可直接参考的成熟项目和实现思路。

推荐阅读顺序：

1. `01-需求与结论/01-项目定位与关键结论.md`
2. `02-系统架构/01-太阳能低功耗电流管理架构.md`
3. `03-功耗预算/01-10分钟采集上传功耗预算.md`
4. `04-采购方案/01-推荐采购方案与成本核算.md`
5. `05-成熟方案/00-成熟方案索引.md`
