# 01-DOIT-ESP32-DEVKIT-V1-30P

这是当前接在电脑 `COM84` 上的 ESP32 开发板资料目录。

基于照片中的丝印、引脚排列、`EN/BOOT` 按键位置、`Micro USB + CP2102` 组合，以及本机读取到的芯片信息，
这块板子可较高概率识别为：

- `DOIT ESP32 DevKit V1 30-pin`
- 或与其完全兼容的同版克隆板

目录说明：

- `01-板卡识别`：根据照片和本机检测结果做出的型号判断
- `02-原理图`：当前板卡最贴近的引脚图与参考原理图
- `03-资源`：芯片、模组、硬件设计与板卡页面资料
- `04-成熟项目`：可直接参考的项目/仓库说明
