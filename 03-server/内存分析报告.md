## 四、一键关闭命令

```bash
systemctl stop openclaw-gateway
systemctl stop jdog
systemctl stop MonitorPlugin
systemctl stop ifrit-agent
systemctl stop JCSAgentCore
```

若要永久禁用（重启后不再启动）：

```bash
systemctl disable openclaw-gateway
systemctl disable jdog 2>/dev/null || true
systemctl disable MonitorPlugin
systemctl disable ifrit-agent
# 不建议 disable，保留控制台重置密码/重装系统能力
```

> 建议先只停止 `openclaw-gateway` 观察效果，再决定是否禁用其他 Agent。
